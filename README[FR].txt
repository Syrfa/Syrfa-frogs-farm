"Syrfa frogs farm" est un script, qui a pour objectif de farmer les albinauric.

site de grâce :

"CORNICHE MENANT AU PALAIS"

voici une vidéo de démonstration : 


les touches devront être changé dans le script si elles sont différentes, sinon,   
changer ses propres touches sur elden ring :

z : avancer
s : reculer
t : compétences 
g : ouvrir la map
f : sauter 
j : déplacement de la caméra (gauche) 

le script se lancera avec un double clique sur le fichier "Syrfa frogs farm.ahk",
ouvrir Elden ring.
Appuyer sur "F3" pour lancer le script et "F8" pour l'arrêter.


Calcul pour connaître le nombre de runes par heure (NG+X):

temps du script : 21 secondes
nombre de runes par run (exemple) : 60 000 runes

3600/21 = 171.42

171.42*60 000 = 10 258 200 par heure.
